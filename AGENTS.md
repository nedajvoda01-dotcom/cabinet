# AGENTS.md — правила по дизайну

## 0) Формат правок
- Любые изменения в `index.html` и других файлах выдавай ТОЛЬКО заменой целых BLOCK-ов.
- В ответе пиши: `BLOCK N) <название>` и ниже весь блок целиком.
- Никаких частичных диффов, кусочков кода или “вставь строку туда-то”.

## 1) Палитра (утвержденные цвета)

### Базовые
- Black / Primary: #000000 (RGB 0,0,0)
- White: #F9F9F9 (RGB 249,249,249)
- Dark Gray: #333333 (RGB 51,51,51)
- Gray: #646464 (RGB 100,100,100)
- Light Gray: #A7A7A7 (RGB 167,167,167)

### Акцентные (только по команде владельца проекта)
- Brand Orange: #E85002 (RGB 232,80,20)

### Gradient stops (ровно эти 4, без изменений)
- #000000
- #C10801
- #F16001
- #D9C3AB

## 2) Принципы использования цветов
- Не добавляем новые HEX/RGB без согласования.
- В CSS/HTML не пишем цвета руками, используем токены (`--c-*`, `--g-*`).
- Белый текст на чёрном; чёрный/тёмный текст на белом/светло-сером; избегаем “серый на сером”.

## 3) Роли цветов
- Основной фон страниц: White #F9F9F9; меню/тёмные блоки: Black #000000; разделение зон: Light Gray #A7A7A7 умеренно.
- Текст на светлом фоне: Black #000000; вторичный текст: Gray #646464; третичный/disabled: Light Gray #A7A7A7; текст на тёмном фоне: White #F9F9F9; подчёркивания/детали: Dark Gray #333333.
- Границы: на тёмном фоне Border #2a2a2b или Dark Gray #333333; на светлом фоне Light Gray #A7A7A7 или Gray #646464.
- Hover тёмных элементов: Dark Gray #333333; active/selected: Gray #646464. На светлом фоне hover/active только серыми.

## 4) Токены в коде
```css
:root{
  --c-black: #000000;
  --c-white: #F9F9F9;
  --c-gray-dark: #333333;
  --c-gray: #646464;
  --c-gray-light: #A7A7A7;

  /* акценты (только по разрешению) */
  --c-orange: #E85002;
  --g-1: #000000;
  --g-2: #C10801;
  --g-3: #F16001;
  --g-4: #D9C3AB;
}
```

Дальше везде только так:

background: var(--c-black);

color: var(--c-white);

border-color: var(--c-gray-dark);
