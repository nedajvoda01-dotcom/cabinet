# AGENTS.md — постоянные правила проекта

## 0) Формат правок
- Любые изменения в `index.html` и других файлах выдавай ТОЛЬКО заменой целых BLOCK-ов.
- В ответе пиши: `BLOCK N) <название>` и ниже весь блок целиком.
- Никаких частичных диффов, кусочков кода или “вставь строку туда-то”.

## 1) Палитра (утвержденные цвета)

### Базовые (можно использовать везде):
- Black / Primary: #000000 (RGB 0,0,0)
- White: #F9F9F9 (RGB 249,249,249)
- Dark Gray: #333333 (RGB 51,51,51)
- Gray: #646464 (RGB 100,100,100)
- Light Gray: #A7A7A7 (RGB 167,167,167)

### Акцентные (только по команде владельца проекта):
- Brand Orange: #E85002 (RGB 232,80,20)

### Gradient stops (ровно эти 4, без изменений):
- #000000
- #C10801
- #F16001
- #D9C3AB

## 2) Принципы использования

### 2.1 Никаких “левых” цветов
- Запрещено добавлять другие HEX/RGB (даже “почти такие же”).
- Если нужен новый оттенок — сначала согласование, потом добавляем как токен.

### 2.2 Только токены
- В CSS/HTML не пишем цвета руками.
- Используем переменные (`--c-*`, `--g-*`), чтобы менять тему централизованно.

### 2.3 Контраст и читабельность
- Белый текст на чёрном.
- Чёрный/тёмный текст на белом/светло-сером.
- Не делаем “серый на сером”, если текст теряется.

## 3) Роли цветов (где какой)

### 3.1 Фон
- Основной фон страниц: White #F9F9F9
- Фон меню/тёмных блоков: Black #000000
- Разделение зон (если надо): Light Gray #A7A7A7 очень умеренно.

### 3.2 Текст
- Основной текст на светлом фоне: Black #000000
- Вторичный текст/подписи: Gray #646464
- Третичный/disabled: Light Gray #A7A7A7
- Текст на тёмном фоне: White #F9F9F9
- Подчёркивания/неважные детали: Dark Gray #333333

### 3.3 Границы/разделители
- На тёмном фоне: Border #2a2a2b (или Dark Gray #333333, но не смешивать без причины).
- На светлом фоне: Light Gray #A7A7A7 или Gray #646464 (по контрасту).

### 3.4 Hover / Active
- Hover тёмных элементов: Dark Gray #333333 (или текущий #141414 без добавления новых оттенков).
- Active/selected: Gray #646464 (или текущий #303030 без добавления новых оттенков).
- На светлом фоне hover/active только серыми, никаких других цветов.

## 4) Использование оранжевого и градиента (строго ограничено)

### 4.1 Оранжевый #E85002
- НЕ используем по умолчанию.
- Используем только там, где это явно разрешено владельцем проекта.
- Если оранжевый включён — он единственный акцент в блоке.

### 4.2 Градиент
- Используется только в разрешённых местах.
- Нельзя ставить на фон всего приложения/карточек без решения владельца.
- Всегда ровно 4 стопа `--g-1..--g-4`, без изменений.

## 5) Токены в коде
- В `:root` должны быть только эти токены:

```css
:root{
  --c-black: #000000;
  --c-white: #F9F9F9;
  --c-gray-dark: #333333;
  --c-gray: #646464;
  --c-gray-light: #A7A7A7;

  /* акценты (только по разрешению) */
  --c-orange: #E85002;
  --g-1: #000000;
  --g-2: #C10801;
  --g-3: #F16001;
  --g-4: #D9C3AB;
}


Дальше везде только так:

background: var(--c-black);

color: var(--c-white);

border-color: var(--c-gray-dark);

6) Логика “Избранного”

Кнопка bookmark должна быть без контейнера и всегда находиться справа от меню, в белом блоке.

Страницы cars, content, и 404 — всегда disabled для добавления в избранное.

Цвета кнопки:

disabled = серый (var(--c-gray-light))

enabled = чёрный (var(--c-black))

active = оранжевый (var(--c-orange))

Состояние и список избранного хранятся в localStorage.

В меню секция “Закладки” показывает название открытого интерфейса, если он добавлен.

7) Тестирование (обязательное)

После любых изменений ОБЯЗАТЕЛЬНО выполняй самопроверку:

Логически пройди сценарии и опиши результат.

Проверь потоки:

загрузка страниц по PAGE_PATHS

404 показывается только если файл по пути реально не найден

bookmark: disabled/enabled/active

сохранение/чтение localStorage

корректный рендер секции "Закладки"

поведение при свернутом/развернутом меню

Если среда позволяет запуск — запусти локально и опиши, что увидел.

Если запуск невозможен — сделай статическую проверку по коду и явно объясни, почему сценарии работают.

Запрещено завершать задачу без блока “Проверка” (OK / найдено что-то).
